
HPDividerStandard

---

Draws a `/` graphic that divides a unit's current and maximum HP.

It uses a palette and tiles that are already in PalRAM/VRAM when the window is built.

---

Definitions:

  * `HP_DIVIDER_X`: The X position in tiles
  * `HP_DIVIDER_Y`: The Y position in tiles
  * `HP_DIVIDER_BASE_TILE`: The VRAM tile index of the first tile of the label graphics
  * `HP_DIVIDER_PALETTE`: The PalRAM index of the label's palette
